# Food Donation App

A full stack food donation application using Flutter and Spring Boot.

## Tech Stack

### Frontend
- Flutter

### Backend
- Spring Boot
- Java

### Database
- MySQL

---

## Project Structure

Frontend TA Food donation/
Backend TA Food donation/
Contract API Tugas Akhir/

---

## Run Frontend

```bash
flutter pub get
flutter run
```

## Run Backend

```bash
mvnw.cmd spring-boot:run
```

---

## Database

Import SQL database using phpMyAdmin.

---

## Features

- Authentication
- Donation Posting
- Claim Donation
- Notification
- History
